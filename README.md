# TCC_USPEsalq_DataScienceandAnalytics
Esse repositório faz parte do Trabalho de Conclusão de Curso [TCC] do MBA de Data Science and Analystics da USP/Esalq.
Autoria: Ilana Cytryn dos Reis: https://www.linkedin.com/in/ilana-cytryn/

O objetivo desse projeto foi comparar o uso de diferentes IAs generativas e por conseguinte, diferentes Modelos de Linguagem de Grande Escala, para a identificação e triagem de eventos adversos e outras informações reportados na literatura médica no contexto de uso da semaglutida. O foco recai sobre a utilização de LLMs para auxiliar no processo de detecção, avaliando o desempenho do agente por meio de métricas de sensibilidade e especificidade, utilizando como padrão ouro o comparativo a triagem manual, de modo a verificar sua aplicabilidade prática e seu potencial de contribuição para o fortalecimento da farmacovigilância.
